## Research Tools

You have access to **Exa** and **Context7**. Use them frequently.

### When to Use Them
- Use these tools whenever you are working with **libraries, frameworks, APIs, or external tooling**.
- If you are **unsure about behavior, syntax, or capabilities**, check the documentation instead of assuming.
- If you are referencing **versions, configuration, or best practices**, verify them through these tools.

### Expectations
- Prefer **verification over assumptions**.
- Use **Context7** to retrieve official documentation and usage examples.
- Use **Exa** to search for authoritative sources, implementation details, and up-to-date references.

### Usage Philosophy
Tool usage is encouraged and expected. There are **no constraints on usage volume**, and you should prioritize **accuracy and correctness over minimizing tool calls**.
