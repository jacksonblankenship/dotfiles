# dotfiles

All of my important dotfiles are tracked in this repository. I use a strategy originally proposed by StreakyCobra in [this](https://news.ycombinator.com/item?id=11070797) Hacker News post.

> "No extra tooling, no symlinks, files are tracked on a version control system, you can use different branches for different computers, you can replicate you configuration easily on new installation."

I've outlined a detailed installation guide in [this gist](https://gist.github.com/jacksonblankenship/add88f8d05447f09008974a90cb178db#terminal) if you want to learn more.
